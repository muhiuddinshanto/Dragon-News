# 📰 News Portal (Next.js Project)

A modern news web application built with Next.js (App Router).  
It provides category-based news filtering, dynamic routing, and a clean responsive UI.

---

## 🚀 Live Demo
https://dragon-news-neon.vercel.app/

---

## 📌 Features

- 🗂️ Category-based news filtering  
- 📰 Dynamic news details page  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- ⚡ Fast performance with Next.js App Router  
- 🧭 Sidebar navigation system  
- 🔍 SEO friendly pages using metadata  
- 🎨 Clean UI with Tailwind CSS  

---

## 🛠️ Tech Stack

- Next.js 13+
- React.js
- Tailwind CSS
- React Icons
- Programming Hero News API
